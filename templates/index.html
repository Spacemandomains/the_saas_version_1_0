AI CPO Agent — Gemini MVP

Use the API via Swagger and an API key. This page is just a sanity check that the server is running.

Quick start

Open Swagger:

GET /docs

Flow:

1) POST /auth/signup  -> get api_key
2) POST /me/product-brief -> save your product context
3) POST /api/agent -> generate prd|roadmap|sprint

Example request

Add header: Authorization: Bearer YOUR_API_KEY

{
  "doc_type": "roadmap",
  "title": "Q2 Roadmap",
  "inputs": {
    "quarters": ["Q2"],
    "themes": ["Activation", "Retention"]
  }
}
